# RaccoonBinator
Recombine orbital parameters of binaries generated by [MCLUSTER](https://academic.oup.com/mnras/article/417/3/2300/1095711) and implement mass-dependent multiplicity fractions from [Moe & Di Stefano 2017](https://iopscience.iop.org/article/10.3847/1538-4365/aa6fb6/meta).

## Introduction
The prescriptions in the code were first introduced in [Torniamenti et al. 2021](https://academic.oup.com/mnras/article/507/2/2253/6346558?login=false). In this version, we have generalized them to other orbital parameter distributions and initial mass functions, and we have introduced a prescription for intermediate-mass stars. The code is entirely written in `python`.
